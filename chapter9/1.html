动态内存管理基础

动态内存管理基础

  • 栈内存 堆内存 V.S.
    • 栈内存的特点：更好的局部性，对象自动销毁
    • 堆内存的特点：运行期动态扩展，需要显式释放
  • 在 C++ 中通常使用 new 与 delete 来构造、销毁对象
#include <iostream>

int* fun()
{
    int *res = new int(2);
    return res;
}

int main()
{ 
    int* y = fun();
    std::cout << *y << std::endl; // 2

    delete y;
    std::cout << *y << std::endl; // 0
}
  • 对象的构造分成两步：分配内存与在所分配的内存上构造对象；对象的销毁与之类似

  • new 的几种常见形式

    • 构造单一对象 / 对象数组

      int* x = new int[3]{1, 3, 54};
std::cout << *(x + 2) << std::endl; // 54

delete[] x;

    • nothrow new

      #include <iostream>
#include <new>

int main()
{ 
  int* x = new (std::nothrow) int[3]{1, 3, 54};
  if (y == nullptr)
  {
      // ... 分配不成功逻辑
  }
  std::cout << *(x + 2) << std::endl; // 54
  delete[] x;
}

    • placement new

      #include <iostream>
#include <new>

int main()
{ 
  char ch[sizeof(int)];
  int* x = new (ch) int(4); // 提供了一块 ch 内存
  std::cout << *x << std::endl; // 4
}

    • new auto

      int* x = new int(3);
int* y = new auto(3);

  • new 与对象对齐

#include <iostream>
#include <new>

struct alignas(256) Str{}; // 字节对齐 256

int main()
{ 
    Str* ptr = new Str(); // 开辟的内存首地址是 256 的整数倍
    std::cout << ptr << std::endl;
}
  • delete 的常见用法
    • 销毁单一对象 / 对象数组
    • placement delete：销毁对象但是不把对象对应的内存归还给系统
#include <iostream>
#include <new>

int main()
{ 
    int* x = new int(3);
    delete x;

    int* ptr = new int[5];
    delete[] ptr;
}

  • 使用 new 与 delete 的注意事项

    • 根据分配的是单一对象还是数组，采用相应的方式销毁
    • delete nullptr

    • 不能delete 一个非 new 返回的内存

      int x;
delete &x; // 不允许

    • 同一块内存不能 delete 多次

      #include <iostream>
#include <new>

int main()
{ 
  int* x = new int(3);
  std::cout << x << std::endl;
  delete x;
  std::cout << x << std::endl;
  delete x;
  // 0x55e7f6010e70
  // 0x55e7f6010e70
  // free(): double free detected in tcache 2
  // Aborted (core dumped)

}
