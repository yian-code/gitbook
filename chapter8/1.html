IOStream概述

IOStream概述

  • IOStream 采用流式 I/O 而非记录 I/O ，但可以在此基础上引入结构信息

  • 所处理的两个主要问题

    • 表示形式的变化：使用格式化 / 解析在数据的内部表示与字符序列间转换

      #include <iostream>

int main()
{
  union
  {
      int x;
      int y; // float y;
  };

  x = 100;

  std::cout << x << std::endl; // 100
  std::cout << y << std::endl; // 100 1.4013e-43
}
    • 与外部设备的通信：针对不同的外部设备（终端、文件、内存）引入不同的处理逻辑
  • 所涉及到的操作
    • 格式化 / 解析
    • 缓存
    • 编码转换
    • 传输
  • 采用模板来封装字符特性，采用继承来封装设备特性
    • 常用的类型实际上是类模板实例化的结果

std::basic_ifstream

ios_base <- basic_ios <- basic_istream <- basic_ifstream

对于 fstream

#include <iostream>
#include <fstream>
int main()
{
    std::ifstream x;
}

使用 c++ insight 进行源码探查可得

#include <iostream>
#include <fstream>
int main()
{
  std::basic_ifstream<char> x = std::basic_ifstream<char>();
  return 0;
}
