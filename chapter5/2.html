表达式详述

表达式详述

算术操作符

  • 共分为三个优先级

    • + ， - （一元）
    • *， / ，%
    • + ， - （二元）

  • 均为左结合的

  • 通常来说，操作数与结果均为算数类型的右值；但加减法与一元 + 可接收指针
#include <cstdio>

int main()
{
    int a[3] = {1, 2, 3};
    auto b = a;         // int * b = a;
    auto& c = a;        // int (&c)[3] = a;
    const auto& d = a;  // const int (&d)[3] = a;
    const auto& e = +a; // int *const & e = +a;
}
  • 一元 + 操作符会产生 integral promotion
#include <cstdio>

int main()
{
    short x = 3;
    auto y = +x; // int y = +static_cast<int>(x);
}
  • 整数相除会产生整数，向 0 取整
  • 求余只能接收整数类型操作数，结果符号与第一个操作数相同
  • 满足 (m / n) * n + m % n == m
std::cout << (4 / 3) * 3 + (4 % 3) << std::endl; // 4

逻辑与关系操作符

  • 关系操作符接收算术或指针类型操作数；逻辑操作符接收可转换为 bool 值的操作数
  • 操作数与结果均为右值（结果类型为 bool ）
  • 除逻辑非外，其它操作符都是左结合的
  • 逻辑与、逻辑或具有短路特性
#include <iostream>

int main()
{
    int ptr = nullptr;
    if (ptr && (*ptr == 3)) // ptr 为空则不会对后面的 (*ptr == 3) 进行处理（短路）
    {
        // ...
    }
}
  • 逻辑与的优先级高于逻辑或：a || b && c
  • 通常来说，不能将多个关系操作符串连
std::cout << (5 > 4 > 3) << std::endl; // 0
// 1. 5 > 4    -> True
// 2. True > 3 -> 1 > 3
// 3. 1 > 3    -> 0
  • 不要写出 val == true 这样的代码
#include <iostream>

int main()
{
    int a = 2;
    if (a == true) //等于：if (a == 1) 
    {
        std::cout << a << std::endl;
    }
    else 
    {
        std::cout << "error" << std::endl;
    }
}
  • Spaceship operator: <=>
    • strong_ordering：auto res = (3 <=> 5)
    • weak_ordering
    • partial_ordering：auto res = (3.0 <=> 5.0)
#include <iostream>

int main()
{
//  if (a > b) {        
//  }
//  else if (a < b) {  
//  }
//  else {   
//  }

    int a = 2;
    int b = 3;

    auto res = (a <=> b) // std::strong_ordering res = (a <=> b);
    if (res > 0) {    
    }
    else if (res < 0) {    
    }
    else if (res == 0) {   
    }
}

位操作符

  • 接收右值，进行位运算，返回右值
  • 除取反外，其它运算符均为左结合的
#include <iostream>

int main()
{
    signed char x = 3; // 0000 0011
    signed char y = 5; // 0000 0101

    std::cout << ~x << std::endl; // 1111 1100 -> -4
    std::cout << (x & y) << std::endl; // 0000 0001 -> 1
    std::cout << (x | y) << std::endl; // 0000 0111 -> 7
    std::cout << (x ^ y) << std::endl; // 0000 0110 -> 6
}
  • 注意计算过程中可能会涉及到 integral promotion
signed char x = 3;
auto y = ~x; // int y = ~static_cast<int>(x);

signed char z = 4;
auto t = x & z; // int t = static_cast<int>(x) & static_cast<int>(z)
  • 注意这里没有短路逻辑
  • 移位操作在一定情况下等价于乘（除） 2 的幂，但速度更快
#include <iostream>

int main()
{
    signed char x = 3; // 0000 0011
    std::cout << (x << 1) << std::endl; // 0000 0110 -> 6
    std::cout << (x >> 1) << std::endl; // 0000 0001 -> 1

    signed char y = -4; // 1111 1100
    std::cout << (y << 1) << std::endl; // 1111 1000 -> -8
    std::cout << (y >> 1) << std::endl; // 1111 1110 -> -2
}
  • 注意整数的符号与位操作符的相关影响
    • integral promotion 会根据整数的符号影响其结果
    • 右移保持符号，但左移不能保证
#include <iostream>

int main()
{
    unsigned char x = 0xff; // 1111 1111
    auto y = ~x;
    std::cout << y << std::endl; // -256

    signed char a = 0xff;
    auto b = ~a;
    std::cout << b << std::endl; // 0
}

赋值操作符

  • 左操作数为可修改左值；右操作数为右值，可以转换为左操作数的类型
  • 赋值操作符是右结合的
  • 求值结果为左操作数
  • 可以引入大括号（初始化列表：x = { y };）以防止收缩转换（ narrowing conversion ）
  • 小心区分 = 与 ==
  • 复合赋值运算符
#include <iostream>

int main()
{
    int x = 2;
    int y = 3;
    // 交换两值并不引入额外的内存
    x^=y^=x^=y;
    std::cout << x << y << std::endl;// 3 2
    /*
            x = 2, y = 3
    x^=y    x = 2^3, y = 3
    y^=x    x = 2^3, y = 3^2^3 = 0^2 = 2
    x^=y    x = 2^3^2 = 0^3 = 3, y = 2
    */
}

自增与自减运算符

  • ++; --
  • 分前缀与后缀两种情况
  • 操作数为左值；前缀时返回左值；后缀时返回右值
  • 建议使用前缀形式
#include <iostream>

int main()
{
    int x = 3;
    int y;
    y = x++;

    std::cout << x << std::endl; // 4
    std::cout << y << std::endl; // 3

    int a = 3;
    int b;
    b = ++a;

    std::cout << a << std::endl; // 4
    std::cout << b << std::endl; // 4
}

其它操作符

  • 成员访问操作符： . 与 ->
    • -> 等价于 (*).
    • . 的左操作数是左值（或右值），返回左值（或右值 xvalue ）
    • -> 的左操作数指针，返回左值
#include <iostream>

struct Str
{
    int x = 1;
};

int main()
{
    Str a;
    std::cout << a.x << '\n'; // 1

    Str* ptr = &a;
    std::cout << (*ptr).x << '\n'; // 1
    std::cout << ptr -> x << '\n'; // 1
}
  • 条件操作符
    • 唯一的三元操作符
    • 接收一个可转换为 bool 的表达式与两个类型相同的表达式，只有一个表达式会被求值
    • 如果表达式均是左值，那么就返回左值，否则返回右值
    • 右结合
#include <iostream>

int main()
{
    int x = 1;
    int y = 2;
    true ? (++x) : (++y);
    std::cout << x << '\n'; // 2
    std::cout << y << '\n'; // 2

    // false ? 1 : "Hello";
    // error: operands to ?: have different types 'int' and 'const char*'

    // 右结合
    int score = -10;
    int res = (score > 0) ? 1 : (score == 0) ? 0 : -1;
    std::cout << res << '\n'; // -1
}

  • 逗号操作符

    • 确保操作数会被从左向右求值：std::cout << (2, 3); // 3
    • 求值结果为右操作数
    • 左结合

  • sizeof 操作符

    • 操作数可以是一个类型或一个表达式
    • 并不会实际求值，而是返回相应的尺寸
std::cout << sizeof(int); // 4

int* ptr = nullptr; // *ptr 不能直接求值的
std::cout << sizeof(*ptr); // 4
  • 域解析操作符 ::
#include <iostream>

int x = 4;

int main()
{
    int x = 5;
    std::cout << x << '\n'; // 5
    std::cout << ::x << '\n'; // 4
}
