数组

数组

数组：将一个或多个相同类型的对象串连到一起，所组成的类型

数组的初始化方式：

  • 缺省初始化 int a[2];
  • 聚合初始化 int b[2] = {1, 2};
#include <iostream>
#include <type_traits>

int main()
{
    int b[3] = {}; // 初始化为零
    int c[3] = {1, 2}; // 1, 2, 0
    int d[] = {1, 2, 3};

    std::cout << std::is_same_v <decltype(d), int[3]> << std::endl; // 1
}

注意

  1. 不能用 auto 来声明数组类型
#include <iostream>
#include <typeinfo>

int main(int argc, char **argv)
{   
    auto a = {1, 3, 4};
    std::cout << typeid(a).name() << std::endl;
    // St16initializer_listIiE
}

cd Debug && ./demo | c++filt -t ：std::initializer_list<int>

这个就是使用 auto 定义的数组类型

  1. 数组不能复制
  2. 元素个数必须是一个常量表达式（编译期可计算的值）
int a; // int
int b[10]; // int[10]

int x;
int c[x];  
// int[x] 可变长度数组不支持，有些编译器的扩展可以，但是程序的可扩展性就丧失了
  1. 字符串数组的特殊性（字符串以 \0 来标志着字符串结束）
char str[] = "Hello";
char str2[] = {'H', 'e', 'l', 'l', 'o', '\0'};
std::cout << std::is_same<decltype(str), char[6]>::value << std::endl;

数组的复杂声明

  • 指针数组与数组的指针
#include <iostream>
#include <type_traits>
#include <typeinfo>

int main(int argc, char **argv)
{   
    int x1;
    int x2;
    int x3;
    int* a[3] = {&x1, &x2, &x3};
    std::cout << std::is_same<decltype(a), int*[3]>::value << std::endl; // 1

    int (*b)[3];
    std::cout << std::is_same<decltype(b), int(*)[3]>::value << std::endl; // 1
}
  • 声明数组的引用
#include <iostream>
#include <type_traits>

int main(int argc, char **argv)
{    
    int b[3];
    int (&a)[3] = b;
    std::cout << std::is_same<decltype(a), int(&)[3]>::value << std::endl;
}

数组中的元素访问

  • 数组对象是一个左值
#include <iostream>
#include <type_traits>

int main(int argc, char **argv)
{    
    int a[3] = {1, 2, 3}; // a: l-value
    std::cout << std::is_same<decltype((a)), int(&)[3]>::value << std::endl; // 1
    std::cout << std::is_same<decltype(a), int[3]>::value << std::endl; // 1

    const int b = 3;
    std::cout << std::is_same<decltype((b)), const int&>::value << std::endl; // 1
}
  • 使用时通常会转换成相应的指针类型
  • x[y] → *((x) + (y))
#include <iostream>
#include <type_traits>

int main(int argc, char **argv)
{    
    int a[3] = {1, 2, 3}; 
    auto b = a;
    std::cout << std::is_same<decltype(b), int*>::value << std::endl; // 1
    std::cout << *b << std::endl; // 1
    std::cout << b << std::endl; // 0x7ffc442e822c
    std::cout << &(a[0]) << std::endl; // 0x7ffc442e822c

    std::cout << 1[a] << std::endl; // 2
    std::cout << a[1] << std::endl; // 2

    std::cout << *(b + 1) << std::endl; // 2 这个可以解释为什么数组的下标是零
}

从数组到指针

数组到指针的隐式转换

  • 使用数组对象时，通常情况下会产生数组到指针的隐式转换
  • 隐式转换会丢失一部分类型信息
#include <iostream>

int main(int argc, char **argv)
{    
    int a[3] = {1, 2, 3}; // int[3]
    std::cout << a[0] << std::endl;

    auto b = a; // int*
}
  • 可以通过声明引用来避免隐式转换
auto& b = a; // int(&)[3]

  120          64~67  68~71  72~75  76~79
  64    -->      0      1      2      3
pointer                  array
  • 注意：不要使用 extern 指针来声明数组
    • Unknown Bounded Array 声明
#include <iostream>

extern int* array;

int main(int argc, char **argv)
{    
    std::cout << array << std::endl;
}

获得指向数组开头与结尾的指针 : std::(c)begin, std::(c)end

#include <iostream>
#include <type_traits>

int main(int argc, char **argv)
{    
    int a[3] = {1, 2, 3};
    std::cout << a + 3 << ' ' << &(a[3]) << std::endl;
    // 0x7fff45f0e118 0x7fff45f0e118

    std::cout << std::begin(a) << std::endl; // 0x7ffdd096bdec
    std::cout << std::end(a) << std::endl; // 0x7ffdd096bdf8

    std::cbegin(a) // const int*
    std::begin(a)  // int*
}

其他操作

求元素的个数
  • sizeof 方法
  • std::size 方法，在 gcc 8.2.0 中不是 std 的成员
  • (c)end - (c)begin 方法
#include <iostream>

int main(int argc, char **argv)
{    
    int a[3];

    std::cout << sizeof(a) << std::endl; // 12
    std::cout << sizeof(int) << std::endl; // 4
    std::cout << sizeof(a) / sizeof(int) << std::endl; // 3

    std::cout << std::size(a) << std::endl;

    std::cout << std::end(a) - std::begin(a) << std::endl; // 3
}
元素遍历
  • 基于元素个数
  • 基于 (c)begin/(c)end
#include <iostream>

int main(int argc, char **argv)
{
    int a[4] = {2, 3, 4, 7};

    auto ptr = std::cbegin(a);
    while (ptr != std::cend(a))
    {
        std::cout << *ptr << std::endl;
        ptr = ptr + 1;
    }
}
  • 基于 range-based for 循环（语法糖）
#include <iostream>

int main(int argc, char **argv)
{
    int a[4] = {2, 3, 4, 7};

    for (int x : a)
    {
        std::cout << x << std::endl;
    }
}

C 字符串

C 字符串本质上也是数组

C 语言提供了额外的函数来支持 C 字符串相关的操作 ： strlen, strcmp…

#include <iostream>
#include <cstring>

int main(int argc, char **argv)
{
    char str[] = "Hello";
    auto ptr = str; // char*

    std::cout << strlen(str) << std::endl; // 5
    std::cout << strlen(ptr) << std::endl; // 5

    char str2[] = {'H', 'e', 'l', 'l', 'o'};
    std::cout << strlen(str2) << std::endl; // 10

    char str3[] = {'H', 'e', 'l', 'l', 'o', '\0'};
    std::cout << strlen(str3) << std::endl; // 5
}

[!TIP] 使用 strlen 来计算 {'H', 'e', 'l', 'l', 'o'} 的结果是不确定的，因为它是看到 \0 才算结尾，不然会一直往上累计。

多维数组

  • 本质：数组的数组
    • int a[3][4];
#include <iostream>
#include <type_traits>

int main(int argc, char **argv)
{
    int x1[3];

    int x2[3][4];
    // (int int int int) (int int int int) (int int int int)

    std::cout << sizeof(int) << std::endl; // 4
    std::cout << sizeof(x2[0]) << std::endl; // 16
    std::cout << std::is_same<decltype(x2[0]), int(&)[4]>::value << std::endl; // 1

    int x3[4][3];
    // (int int int) (int int int) (int int int) (int int int)
}
  • 多维数组的聚合初始化：一层大括号 V.S. 多层大括号
int x1[3][4] = {1, 2, 3, 4, 5};
int x2[3][4] = {{1, 2, 3, 4}, {4, 5, 6}};
  • 多维数组的索引与遍历
    • 使用多个中括号来索引
    • 使用多重循环来遍历
#include <iostream>

int main(int argc, char **argv)
{
    int x1[3][4] = {1, 2, 3, 4, 5};
    for (auto& x : x1) 
    // 不使用引用则 x 会隐式转换为指针类型而无法使用 range-based for 循环
    // 可以将其放入 C++ Insights 中查看原因
    {
        for (auto y : x)
        {
            std::cout << y << std::endl;
        }
    }
}

指针与多维数组

  • 多维数组可以隐式转换为指针，但只有最高维会进行转换，其它维度的信息会被保留
#include <iostream>

int main()
{
    int x[3][4];
    auto ptr = x; // int (*ptr)[4] = x;

    int x2[3][4][5];
    auto ptr1 = x2; // int (*ptr1)[4][5] = x2;
    auto ptr2 = ptr1[0]; // int (*ptr2)[5] = ptr1[0];
}
  • 使用类型别名来简化多维数组指针的声明
#include <iostream>

using A2 = int[4][5];

int main()
{
    int x[3][4][5];
    A2* ptr = x; // A2 (*ptr) = x;
    auto ptr1 = ptr[0]; // int (*ptr1)[5] = ptr[0];

    A2 x1[3];
}
  • 使用指针来遍历多维数组
#include <iostream>

int main()
{
    int x2[3][4] = {};
    auto ptr = std::begin(x2);
    while (ptr != std::end(x2))
    {
        auto ptr2 = std::begin(*ptr);
        while (ptr2 != std::end(*ptr))
        {
            std::cout << *ptr2 << std::endl;
            ptr2 += 1;
        }
        ptr += 1;
    }
}
