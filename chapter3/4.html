类型别名与类型的自动推导

类型别名与类型的自动推导

可以为类型引入别名，从而引入特殊的含义或便于使用（如： size_t）。两种引入类型别名的方式

  • typedef int MyInt;
  • using MyInt = int; (从 C++11 开始)
#include <iostream>

typedef int Myint;

int main()
{
    Myint x = 3;
}

使用 using 引入类型别名更好

  • typedef char MyCharArr[4];
  • using MyCharArr = char[4];

[!TIP] 类型别名与指针、引用的关系： 应将指针类型别名视为一个整体，在此基础上引入常量表示指针为常量的类型 不能通过类型别名构造引用的引用

类型的自动推导

  • 从 C++11 开始，可以通过初始化表达式自动推导对象类型
  • 自动推导类型并不意味着弱化类型，对象还是强类型

自动推导的几种常见形式

  • auto：最常用的形式,但会产生类型退化
auto x = 3;
auto z(3.5);

auto y; // error: declaration of 'auto y' has no initializer

类型退化：

int x = 3;
int& ref = x;

int y = ref;  // int& 类型退化为 int 类型

int a = 5;
const int& rex = a;

int b = rex;  // const int& -> int

以下例子证明了 auto 的类型退化

#include <iostream>
#include <type_traits>
int main()
{
    int x = 4;
    int& ref = x;

    auto req = ref; // int& -> int
    std::cout << std::is_same_v<decltype(req), int> << std::endl; // 1
}
  • const auto / constexpr auto： 推导出的是常量 / 常量表达式类型
  • auto& ： 推导出引用类型,避免类型退化
#include <iostream>
#include <type_traits>
int main()
{
    const int x = 3;
    auto& y = x;
    std::cout << std::is_same_v<decltype(y), const int&> << std::endl; // 1

    const auto z = x; // x: const int -> int z: const auto -> const int 
    std::cout << std::is_same_v<decltype(y), const int> << std::endl; // 1
}
  • decltype(exp) ：返回 exp 表达式的类型（左值加引用）
  • decltype(val) ：返回 val 的类型
#include <iostream>
#include <type_traits>
int main()
{
    int x = 4;
    int& y1 = x;

    auto y2 = y1;
    decltype(y1) y3 = y1; // 没有类型退化
    std::cout << std::is_same_v<decltype(y2), int> << std::endl; // 1
    std::cout << std::is_same_v<decltype(y3), int&> << std::endl; // 1

    int* ptr = &x; 
    std::cout << std::is_same_v<decltype(*ptr), int&> << std::endl; // 1
    std::cout << std::is_same_v<decltype(ptr), int*> << std::endl; // 1
}
  • decltype(auto) ：从 c++14 开始支持,简化 decltype 使用
  • concept auto：从 C++20 开始支持,表示一系列类型( std::integral auto x = 3; )
#include <iostream>
#include <type_traits>
#include <typeinfo>
#include <concepts>
int main()
{
    decltype(3.5 + 13l) x = 3.5 + 13l;

    decltype(auto) y = 1.2 + 22l;

    std::integral auto z = 3;
    std::cout << std::is_same_v<decltype(z), int> << std::endl; // 1
}
